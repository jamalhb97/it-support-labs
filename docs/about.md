# Sobre mí

Hola! soy **Jamal**,

Este sitio web forma parte de mi proceso de aprendizaje y desarrollo como **Técnico de Soporte Técnico IT**, donde documento **laboratorios reales** de sistemas, redes y resolución de incidencias simulando situaciones que pueden darse en un entorno de trabajo.

## Objetivo del proyecto

El objetivo de esta web es documentar **laboratorios prácticos reales** relacionados con:

- Windows
- Linux
- Redes 
- Active Directory
- Resolución de incidencias IT


Todos los laboratorios están planteados como **situaciones reales** con un enfoque práctico y orientado a la resolución de problemas siguiendo una metodología similar a la que se utiliza en soporte técnico profesional.

## Tecnologías y herramientas

Durante el desarrollo de estos laboratorios trabajo con tecnologías habituales en entornos IT:

- Windows 10 / Windows Server
- Active Directory
- Linux
- Redes (TCP/IP, DNS, DHCP)
- Virtualización
- Git y GitHub
- Documentación técnica con MkDocs


## En aprendizaje continuo

Este proyecto está en **evolución constante**.
A medida que avanzo en mi formación, voy incorporando nuevos laboratorios, mejorando la documentación y ampliando conocimientos técnicos.

El objetivo final es reflejar no solo lo que sé hacer, sino **cómo afronto y resuelvo problemas reales de soporte IT**.
