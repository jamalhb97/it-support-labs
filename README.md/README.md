# IT Support Labs

Repositorio de **laboratorios prácticos de Soporte Técnico IT**, enfocados en situaciones reales que pueden ocurrir en un entorno profesional.

Este proyecto está diseñado como una **base de conocimiento práctica**, documentando incidencias, diagnósticos y soluciones paso a paso.

---

## Sitio web

La documentación completa está disponible en GitHub Pages:

https://jamalhb97.github.io/it-support-labs/

---

## Contenido del proyecto

Actualmente el proyecto incluye laboratorios relacionados con:

- Windows
- Linux
- Redes
- Active Directory
- Resolución de incidencias IT

Cada laboratorio sigue una estructura clara:

1. Descripción de la incidencia  
2. Entorno y diagnóstico inicial  
3. Identificación de la causa  
4. Solución aplicada  
5. Verificación y resultado final  

---

- Windows 10 / Windows Server

## Objetivo

El objetivo de este proyecto es demostrar **capacidades prácticas reales** en soporte técnico IT, más allá de la teoría, mostrando cómo se analizan y resuelven problemas reales.
